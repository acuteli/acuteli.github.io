git介绍

工作区: 在自己电脑上能看到的目录
暂存区: index,一般存放在.git目录下的index文件(.git/index)中,所以暂存区也叫索引
版本库: 工作区一个隐藏目录.git,是Git的版本库

常用命令:

git version 查看安装git的版本
node -v 查看node版本
nmp install -g hexo 安装hexo;

hexo init 初始化
source：用命令创建的各种文章

创建github,建立秘钥 youname.github.io(Public)

git config user.name 查看姓名
git config user.email 查看email设置;

hexo clean 清出缓存;
hexo generate 改变md文档为静态网页;
hexo deploy 上传;
hexo server 启动服务;
