架构

发表于 2017-08-10 | 分类于 base

以下两个是目前网站的主流架构:

LAMP是基于Apache,LNMP是基于Nginx;

LAMP = Linux+Apache+Mysql+PHP
LAMP = Linux+Nginx+Mysql+PHP

Nginx和Apache相比较:

Apache是世界上使用排名第一的web服务器,其跨平台和安全性是最流行的web服务器之一;相对于Nginx,Apache有些臃肿,内存和CPU消耗也比较大;Nginx对于静态文件的响应能力也高于Apache;

Apache是负载PHP的最佳选择,如果流量很大,可以使用Nginx负载非PHP请求

Nginx是对内进行支持,对外进行服务,是一款优秀的邮件代理服务器,安装简单,配置简介.

Nginx和Apache:都是反向代理的web服务器,

反向代理: Nginx比Apache好;

热部署: Nginx支持热部署;Apache不支持热部署;

系统压力: 二者都比较小;

稳定性:Nginx要比Apache好;

安全性: Nginx在Apache之上;

技术资料: Nginx相对较少,Apache比较多;

静态文件处理: Nginx比Apache好;

虚拟主机: 两者都支持;

内存消耗: Nginx非常小,Apache比较大;
